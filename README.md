> 🏆 TRAE AI 创造力大赛 · 学习工作赛道参赛作品

## 项目简介

一款面向短视频创作者的 AI 智能文案辅助工具（Web 端），专注于**深度文案拆解、风格化改写、原创灵感激发与智能文案优化**，帮助创作者解决"灵感枯竭、拆解低效、AI 生成内容同质化"三大痛点。

## 核心功能

### 🔍 爆款文案拆解
一键解析爆款文案的钩子设计、结构逻辑、情绪节奏和传播亮点，让你秒懂"为什么火"。

### ✨ 多平台风格改写
支持抖音、小红书、B站、快手四大平台风格一键切换，同一条文案产出 N 个版本。

### 💡 原创灵感激发
从故事切入、反差对比、干货教程、情感共鸣、热点结合等多个维度生成创意切入点。

### 🎯 智能文案优化
从钩子强度、节奏感、金句设计到结尾引导，全方位优化你的文案。

## 快速开始

### 本地运行
直接用浏览器打开 `video-copy-assistant/index.html` 即可体验。

### 在线体验
Demo 已打包为 ZIP 文件，下载解压后用浏览器打开即可。

## 技术栈

- 纯 HTML + CSS + JavaScript（单文件，无依赖）
- 响应式设计，支持桌面端和移动端
- 深色主题 + 橙红渐变风格
